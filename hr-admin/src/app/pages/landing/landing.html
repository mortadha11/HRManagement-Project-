About HRManage

A practical HR application connected to a real backend

HRManage is built to support the everyday work of an HR department. The application connects the Angular interface to a .NET API and SQL Server database so employees, accounts, departments, contracts, and leave data are stored and updated from the system instead of only changing on the screen.

The current version focuses on the core HR workflow: secure login, role-based navigation, employee CRUD, password management, dashboard summaries, and employee self-service.

Workflow

How the platform works

1
Sign in

The backend validates the username and password and returns a JWT.

2
Route by role

Admins and managers go to the dashboard. Employees go to their profile.

3
Manage data

Dashboard actions call the API and update SQL Server records.

4
Stay secure

Protected API calls use the bearer token stored in the browser session.

Role based

Different spaces for admin, manager, and employee users.

Database driven

Create, update, read, and delete operations are persisted through the API.

Self-service

Employees can maintain their profile and update their password safely.
